Flaskr Plus
Author: Hirat Rahman Rahi

A tiny Flask microblog rebranded from the Flask tutorial, with a few upgrades: categories for posts, filter by category, and per-post delete.

Features

Add posts with a free-text category

Filter the list by category (dropdown) + Clear filter

Delete a post via a button (POST)

Changes in code are marked with FLASKR PLUS CHANGE
